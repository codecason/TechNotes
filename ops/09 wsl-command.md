安装

~~~
wsl --install <DistroName>
~~~

查看

~~~
wsl -l -v
~~~

卸载

~~~
wsl --unregister Ubuntu (<DistroName>，不会完全删除，要在开始菜单-应用设置重置)
~~~

关闭

~~~
wsl --shutdown
~~~

启动

~~~
wsl -d <distroname>
~~~
